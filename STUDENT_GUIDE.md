# Python 코드카타 학생 가이드

Python 코드카타 문제 풀이 결과를 자동으로 GitHub에 업로드하는 크롬 익스텐션입니다.

- 지원 플랫폼: 프로그래머스 (393문제), 백준 (38문제) - 총 431문제

---

### 1. 익스텐션 설치하기

- 익스텐션 파일(python_submit_chrome 폴더)을 다운로드 후 압축 해제합니다.
    - [스크린샷]

- Chrome 브라우저에서 `chrome://extensions` 접속

- 우측 상단 '개발자 모드' 활성화
    - [스크린샷]

- '압축해제된 확장 프로그램을 로드합니다' 클릭 → 압축해제한 폴더 선택
    - [스크린샷]

- 툴바에 고정하기
    - 주소창 우측의 퍼즐 아이콘(확장 프로그램) 클릭
    - 'Python 코드카타' 옆의 핀 아이콘 클릭하여 고정
    - [스크린샷]

---

### 2. 초기 설정하기

2-1. 이름 설정

- Chrome 도구모음에서 익스텐션 아이콘 클릭
- 이름 입력 후 '저장' 버튼 클릭
    - [스크린샷]

2-2. GitHub 로그인

- 'GitHub로 로그인' 버튼 클릭
    - [스크린샷]

- 화면에 표시되는 코드가 자동으로 클립보드에 복사됩니다
    - [스크린샷]

- 새 탭에서 GitHub 인증 페이지가 열리면 코드를 붙여넣기(Ctrl+V)
    - [스크린샷]

- 'Authorize' 버튼 클릭하여 권한 승인
    - [스크린샷]

2-3. 저장소 선택

- 로그인 완료 후 저장소 선택 드롭다운에서 기존 저장소 선택
    - 또는 '+ 새 저장소 만들기 (python-codekata)' 버튼 클릭하여 새 저장소 생성
    - [스크린샷]

### 3. 사용 방법

- 프로그래머스 또는 백준에서 등록된 문제 풀이
    - 프로그래머스: school.programmers.co.kr
    - 백준: www.acmicpc.net

- 문제를 풀고 '정답입니다' 또는 '맞았습니다!!' 판정을 받으면 자동으로 GitHub에 코드가 업로드됩니다
    - [스크린샷]

- 업로드 성공 시 화면 우측 상단에 알림이 표시됩니다
    - [스크린샷]

### 4. 기능 설명

- 문제 목록에서 체크박스(□)를 클릭하면 완료/미완료 전환됩니다
    - 수동으로 풀이 기록을 관리할 수 있습니다
    - [스크린샷]

- 문제 제목을 클릭하면 해당 문제 페이지로 바로 이동합니다
    - [스크린샷]

- '추천' 배지가 있는 다음 문제 클릭 시 해당 문제로 이동합니다
    - [스크린샷]

### 5. 진행 상황 확인

- 상단: 전체 진행률 (예: 45/431)

- 진행률 요약 카드:
    - 전체 %: 전체 완료율
    - 프로그래머스: 프로그래머스 완료 문제 수
    - 백준: 백준 완료 문제 수
    - 기초·입문: L0 레벨 완료 문제 수

- 탭별 진행률:
    - 기초·입문: 코딩테스트 입문(100문제) + 코딩 기초 트레이닝(123문제)
    - 프로그래머스: L1~L5, L7 레벨 문제 (170문제)
    - 백준: L8 레벨 문제 (38문제)

- 각 난이도별 진행률 바로 확인 가능
    - [스크린샷]

### ❓ FAQ (자주 묻는 질문)

Q: 정답인데 자동 업로드가 안돼요

- 등록된 431문제인지 확인해주세요
- GitHub 로그인이 되어 있는지 확인해주세요
- 저장소가 선택되어 있는지 확인해주세요
- 익스텐션 최초 설치 후 페이지를 새로고침 후 다시 시도해주세요

Q: 수동으로 완료 체크할 수 있나요?

- 네! 문제 목록에서 체크박스(□)를 클릭하면 완료/미완료 전환됩니다

Q: 다른 컴퓨터에서도 진행률이 유지되나요?

- 네! Chrome 동기화가 켜져 있으면 Google 계정으로 자동 동기화됩니다

Q: GitHub 저장소 구조는 어떻게 되나요?

- 아래와 같은 구조로 자동 정리됩니다:
```
your-repo/
├── programmers/
│   ├── L0_기초입문/
│   │   ├── 입문/
│   │   └── 기초트레이닝/
│   ├── L1_입문/
│   ├── L2_기초/
│   ├── L3_중급/
│   ├── L4_중상/
│   ├── L5_고급/
│   └── L7_챌린지/
└── baekjoon/
    └── L8_백준/
```
